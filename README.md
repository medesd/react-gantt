## Installation

- Git clone
- `npm install` 

## Run the app in the development mode
- `npm start`

## About

React Gantt Chart is a simple chart that implements gantt functionality as 
a React components.

the project was tested and should work on:

 - Firefox 50+
 - Chrome 49+
 - Safari 10+
 - Edge 14+
 - IE 9+
